# 🩺 Diabetes Prediction Model using SVM

A Machine Learning-based web application that predicts whether a person is likely to have diabetes using the **Support Vector Machine (SVM)** algorithm. This project is built with Python and uses medical diagnostic data to provide accurate and fast predictions.

---

## 🚀 Features

* Predicts diabetes based on medical input parameters
* Uses **SVM (Support Vector Machine)** for classification
* Simple and interactive user interface
* Fast and reliable predictions
* Suitable for integration into healthcare applications

---

## 🧠 Tech Stack

* **Python**
* **Scikit-learn**
* **Pandas & NumPy**
* **Matplotlib / Seaborn** (for visualization)
* **Flask / Streamlit** (for web interface)

---

## 📊 Dataset

The model is trained on a medical dataset (e.g., Pima Indians Diabetes Dataset) with the following features:

* Pregnancies
* Glucose Level
* Blood Pressure
* Skin Thickness
* Insulin Level
* BMI
* Diabetes Pedigree Function
* Age

---

## ⚙️ Project Workflow

1. Data Collection & Preprocessing
2. Handling Missing Values
3. Feature Scaling & Normalization
4. Training the SVM Model
5. Model Evaluation (Accuracy, Precision, Recall)
6. User Input via UI
7. Prediction Output
